OpenAI API Key 발급 및 사용 가이드

LecAI에서 GPT 모델을 사용하기 위해 필요한 API Key를 발급받는 방법을 안내합니다.

1 OpenAI 계정 생성 및 로그인

OpenAI 서비스를 이용하려면 먼저 계정이 필요합니다. 이미 계정이 있다면 로그인해주세요.

공식 사이트 바로가기: OpenAI 회원가입 페이지

2 API Key 생성 페이지 이동

로그인 후, 우측 상단 메뉴 또는 대시보드에서 API Key 관리 페이지로 이동합니다.

  1. 로그인 후 우측 상단의 프로필 아이콘 클릭
  2. 'View API keys' 메뉴 선택
  3. 또는 아래 링크를 클릭하여 바로 이동하세요.
API Key 관리 페이지 바로가기

3 새 API Key 발급 (Create new secret key)

API Key 페이지에서 '+ Create new secret key' 버튼을 클릭합니다.

설정 팁:

  • Name (Optional): 키의 용도를 식별하기 위한 이름입니다 (예: "LecAI Key").
  • Permissions: 기본값인 'All'을 유지하거나 필요한 권한만 설정할 수 있습니다.

중요: 키가 생성되면 팝업창에 단 한 번만 표시됩니다. 반드시 복사하여 안전한 곳에 저장하세요. 창을 닫으면 다시 볼 수 없습니다!

4 결제 수단 등록 (필수)

OpenAI API는 유료 서비스이므로, API Key가 작동하려면 결제 수단(신용카드)이 등록되어 있어야 하고, 잔액(Credit)이 충전되어 있어야 합니다.

결제 설정 페이지 바로가기

* ChatGPT Plus 구독(월 $20)과는 별개입니다. API 사용량에 따라 별도로 과금됩니다. (선불 충전 방식)

5 LecAI 설정에 키 입력

발급받은 키(sk-... 로 시작)를 복사하여 LecAI 설정 페이지의 'OpenAI API Key' 입력창에 붙여넣고 저장하세요.
